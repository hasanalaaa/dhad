# إضافة ضاد للمتصفحات — Manifest V3

هذه الحزمة جاهزة للتحميل غير المعبأ في Chrome أو Edge:

1. شغّل خادم ضاد: `dhad serve`.
2. افتح `chrome://extensions` أو `edge://extensions`.
3. فعّل وضع المطور واختر **Load unpacked** ثم مجلد `extension/`.
4. افتح نافذة ضاد، اترك العنوان المحلي الافتراضي أو أدخل خادم HTTPS موثوقًا، ثم اختبر الاتصال.

## نموذج الأمان والخصوصية

- أذونات المضيف الدائمة محصورة في `localhost` و`127.0.0.1`.
- أي خادم آخر يحتاج موافقة متصفح صريحة عبر `optional_host_permissions`.
- Service Worker وحده ينفذ اتصالات الشبكة، ولا تُحقن مفاتيح أو أسرار في صفحات الويب.
- لا توجد تحليلات استخدام أو تتبع، ولا يُخزن نص المستخدم في الإضافة.
- الاقتراحات الأسلوبية واللهجية لا تُعامل كتعديلات آمنة تلقائيًا؛ زر البطاقة يوضح أنها تحتاج مراجعة.
