# أساس الهاتف في v1.0.0-rc1 — PWA

واجهة `src/dhad/web/` تطبيق ويب تقدمي قابل للتثبيت على Android وiOS وChromeOS:

- `manifest.webmanifest` يحدد اللغة العربية وRTL ووضع `standalone` وأيقونات maskable.
- `service-worker.js` يوفّر غلاف الواجهة عند انقطاع الشبكة، لكنه يتعمد عدم تخزين أي API request/response يحمل نصًا.
- التخطيط متجاوب مع الشاشات الصغيرة، وجميع الإجراءات التي تغير النص تتطلب ضغط المستخدم.
- يمكن تشغيل الخادم على الجهاز نفسه، أو تحديد نشر HTTPS موثوق عبر إعدادات الشبكة/الإضافة.

هذا الأساس ليس لوحة مفاتيح أصلية لنظامي iOS/Android، ولا يدّعي ذلك. تحويله لاحقًا إلى حزمة متجر يمكن أن يستخدم Trusted Web Activity أو Capacitor فوق واجهة PWA الحالية من دون نسخ منطق اللغة.
