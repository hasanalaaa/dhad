# تقرير Phase 5 — الوضوح والأسلوب والنبرة

**الإصدار:** `0.7.0`  
**الحالة:** التنفيذ الهندسي الحتمي مكتمل؛ التقييم البشري الخارجي للمنفعة الأسلوبية لم يُدّعَ إنجازه.

## البنية المنفذة

- أضيف `dhad.style.StyleEngine` فوق مخرجات `MorphologicalAnalyzer` و`SyntaxEngine`، ويعيد `StyleReport` غير متغير يضم الملاحظات والنبرة ومؤشرات الوضوح.
- مورد `style_lexicon.json` متحقق منه بـ`style_lexicon.schema.json`، ويحتوي 16 تفضيلًا عباراتيًا مدققًا، و21 إشارة نبرة، و9 خرائط لتحويل المصدر إلى فعل مباشر.
- يدعم المحرك سبعة Profiles: عام، أكاديمي، إداري، صحفي، تعليمي، ودّي، وأدبي.
- تنتمي الملاحظات إلى خمس عائلات: الحشو، التطويل، القوالب المستهلكة، العبارات المهجورة، والركاكة.
- يعيد بناء «قام/قامت/قاموا + بحرف جر + مصدر» بفعل مباشر مناسب للتأنيث أو الجمع، ثم يتحقق المحلل الصرفي من صيغة الاقتراح قبل عرضه.
- يكشف تراكم المصادر والروابط في الجمل الطويلة من خلال تحليل Phase 4، وينبه إلى تحولات النبرة المحادثية داخل مستند رسمي أو موضوعي مستقر.
- يصنف النبرة إلى: رسمية، موضوعية، حازمة، إقناعية، محادثية، أو محايدة. كل نتيجة تحمل أدلة نصية وإزاحات وأوزانًا قابلة للتفسير.
- يوفّر «مؤشر وضوح ضاد v1» قياسًا هندسيًا شفافًا لطول الجملة والكلمة وكثافة المصادر والتكرار؛ وهو مؤشر مقارنة داخلي وليس قانون قراءة عربيًا معتمدًا.

## السلامة والتكامل

- كل نتيجة جديدة `category=style` و`autofix=false` وتحمل الوسم `requires-approval`.
- `Dhad.correct(..., mode="safe")` لا يطبق أي اقتراح Phase 5. التطبيق لا يحدث إلا بطلب صريح عبر `mode="all"` أو `dhad fix --all`.
- يعاد استخدام `DocumentParse` نفسه الذي أنشأته طبقة Phase 4 داخل خط الفحص، بدل إعادة التحليل الصرفي والنحوي.
- يمكن تعطيل طبقة الأسلوب في `Dhad(style_checks=False)` من دون تعطيل واجهة التقرير المنفصلة.
- أضيفت الواجهات `Dhad.style_report()` و`Dhad.analyze_tone()` والأمر `dhad style --json`.

## التحقق الآلي

- **1,530 اختبارًا ناجحًا**، منها 48 اختبارًا مباشرًا لمحرك Phase 5، إضافة إلى اختبارات عزل وانحدار عبر المراحل.
- Ruff نظيف والتنسيق متحقق، ولا توجد مواضع `pass` أو `TODO` أو `FIXME`.
- Benchmark test مضبوط: 750 جملة و13,226 كلمة.
- مقارنة v0.7.0 بـv0.6.0 على النطاق القياسي نفسه:
  - Precision: **0.8497 → 0.8497**
  - Recall: **0.8764 → 0.8764**
  - F0.5: **0.8549 → 0.8549**
  - FP/1000: **6.3511 → 6.3511**
- عند فصل الفئات الميكانيكية (`spelling/grammar/punctuation`) كانت النتائج متطابقة سواء شُغلت طبقة Phase 5 أم عُطلت، ما يثبت أن اقتراحاتها لا تضخم الإنذارات الميكانيكية.
- قياس محلي دافئ لنص 645 محرفًا عبر 200 تشغيل: P50 **4.26ms** وP95 **4.46ms** لخط الفحص الكامل في بيئة البناء.

## الحد المنهجي

الـBenchmark الحالي مضبوط ومولد من قوالب، ولا يقيس منفعة الصياغة البشرية أو حفظ المعنى في نصوص طبيعية واسعة. لذلك لا تدعي هذه النسخة بلوغ بوابة «80% اقتراحات مفيدة» قبل إجراء مراجعة بشرية مزدوجة مستقلة. ما أُثبت هنا هو اكتمال الكود، التصنيف الآمن، قابلية التفسير، وعدم حدوث انحدار ميكانيكي.
