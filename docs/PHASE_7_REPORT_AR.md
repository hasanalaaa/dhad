# تقرير Phase 7 — الطبقة العصبية الهجينة

**الإصدار:** `0.9.0`  
**الحالة:** التنفيذ الهندسي مكتمل؛ النموذج الافتراضي خفيف ومحلي، ولا توجد ادعاءات دقة عامة لنموذج Transformer غير مدرّب ضمن المستودع.

## البنية المنفذة

- حزمة `dhad/neural/` مستقلة تضم عقد `NeuralBackend`، نموذج n-gram خطي محلي، بوابة ثقة، وواجهة `TransformerBackend` فعلية ذات تحميل كسول لـHuggingFace.
- النموذج الافتراضي مورد JSON متحقق منه بـJSON Schema، ويستخدم سمات نافذة سياقية، وأنواع الكلمات، والتعريف، وموقع الكلمة.
- فك الالتباس مقيد حصريًا بالقراءات التي أنتجها `MorphologicalAnalyzer` أصلًا؛ لا يمكن للطبقة العصبية اختراع تحليل صرفي.
- تُفحص الكلمة فقط إذا كان لها بديلان على الأقل وكان اطمئنان `SyntaxToken` أقل من 0.86 أو كان هامش القراءات الصرفية ≤0.08.
- قبول WSD يحتاج احتمالًا ≥0.78 وهامشًا ≥0.18، ثم يعاد بناء العلاقات والإعراب عبر `SyntaxEngine.rebuild_sentence()` مع الحفاظ على النص والإزاحات.
- التدقيق السياقي للكلمات الصحيحة يحتاج احتمالًا ≥0.985 وهامشًا ≥0.70، ويقتصر على confusion sets موثقة.

## السلامة

- النتائج الاحتمالية مصنفة `neural_suggestion`، و`autofix=false` دائمًا.
- الوضع الآمن لا يطبقها؛ يمكن تطبيقها فقط مع `--all` كقرار صريح.
- القواعد الحتمية والإملاء والنحو واللهجات والأسلوب تملك الأولوية، وأي اقتراح عصبي متداخل يُحذف.
- يمكن تعطيل المرحلة كليًا عبر `Dhad(neural_checks=False)`، وتظل النواة الحتمية كاملة الوظائف.
- لا يحمل التثبيت الأساسي أي نموذج أو مكتبة ثقيلة؛ دعم Transformer اختياري عبر `pip install dhad[neural]`.

## أمثلة مثبتة

- `كتب الطالب الدرس`: إعادة ترتيب القراءة إلى فعل «كتب» من المرشحات الصرفية الموجودة.
- `ثلاثة كتب مفيدة`: اختيار الاسم الجمع «كتاب» بدل الفعل.
- `قابل علم الفيزياء`: اقتراح احتمالي شديد الثقة `عالم`، بلا تصحيح آلي آمن.

## التحقق

- **1,573 اختبارًا ناجحًا** بعد إضافة 13 اختبارًا مباشرًا للطبقة الهجينة.
- Ruff والتنسيق نظيفان.
- Benchmark v0.9.0 مطابق لـv0.8.0: P **0.8602**، R **0.9539**، F0.5 **0.8775**، FP/1000 **6.3511**.
- النطاق الميكانيكي بقي P **0.7551**، R **0.9120**، F0.5 **0.7820**، FP/1000 **6.3511**.

## الحد المنهجي

النموذج الافتراضي صغير ومقصود للتكامل والسلامة، وليس بديلًا عن نموذج عربي مدرّب على corpus طبيعي. واجهة Transformer جاهزة هندسيًا، لكن أي نموذج إنتاجي يجب أن يثبت فائدته على Benchmark مستقل ومراجعة بشرية قبل تفعيله افتراضيًا.
