# تقرير Phase 2 — البيانات والتقييم المستقل

**الإصدار:** `0.4.0`  
**الحالة:** مكتملة هندسيًا ضمن حدود corpus مضبوط ومصرّح بقيوده.

## ما أُنجز

- Schema رسمي لبيانات Benchmark مع تحقق JSON Schema.
- API داخل `dhad.evaluation` للتحميل والتحقق والقياس.
- مقاييس span وcorrection والجملة وF0.5 وFP/1000 words وMRR.
- تقارير sliced حسب المجال واللهجة.
- قياس اتفاق المراجعين: Cohen's kappa وSpan F1 واتفاق التصحيح والحالة الكاملة.
- Corpus مستقل عن أمثلة القواعد بعدد 5,000 جملة، وتقسيم ثابت 3,500/750/750.
- 1,000 حالة بمرورَي تحقق تقنيين مستقلين.
- manifest ببصمات SHA-256 وسجل تراخيص مركزي.
- أمر واحد للتشغيل: `dhad benchmark --split test`.
- تقرير خط أساس محفوظ في `reports/benchmark_v0.4.0_test.*`.

## خط أساس v0.4.0 على test

- الحالات: 750.
- الكلمات العربية: 13,226.
- Span precision: 0.8303.
- Span recall: 0.7583.
- Span F0.5: 0.8148.
- FP لكل 1000 كلمة: 6.3511.
- Sentence exact accuracy: 0.7547.
- MRR للتصحيحات: 1.0000 للحالات التي وجد فيها المحرك النطاق الصحيح.

## القراءة الصحيحة للأرقام

هذه النتائج تكشف مواطن قوة وضعف النسخة الحالية، ولا تُجمّلها. corpus مضبوط اصطناعيًا، لذا يفيد في الانحدار ومقارنة المراحل، لكنه لا يثبت أداءً عامًا على نصوص عربية طبيعية. قبل ادعاء دقة بحثية أو تجارية نحتاج corpus طبيعيًا مرخصًا ومراجعة بشرية مزدوجة حقيقية.

## بوابة الانتقال

المرحلة حققت البنية القابلة للتشغيل، 5,000 حالة، 1,000 مرورَي مراجعة، التقسيم، المقاييس، سجل التراخيص، وخط الأساس المنشور. القيد البشري موثق صراحةً ولا يُخفى.
