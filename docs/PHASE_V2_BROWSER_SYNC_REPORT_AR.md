## م3.2 — المحرك داخل المتصفح (مكتملة)

بُنيت `dhad-core-rs` فعليًا إلى `wasm32-unknown-unknown` بثنائية **1.32MB بعد `wasm-opt -Oz` (471KB مضغوطة)** — دون wasm-bindgen: واجهة C يدوية فوق الذاكرة الخطية (`dc_alloc/dc_free/dc_load_rules/dc_check/dc_tokenize/dc_sentences/dc_normalize`) وجسر JS من ~70 سطرًا (`web_demo/dhad-core.js`).

أُضيف إلى الـcrate محرك قواعد محمول (`src/rules.rs`): 130 قاعدة حرفية صُدّرت من بايثون عبر `tools/export_wasm_rules.py` بنفس دلالات حدود `B_LEFT/B_RIGHT` (فحص المحارف المجاورة يدويًا — Rust regex بلا lookaround)، مع نقل `dedupe` وsweep-line (`src/spans.rs`). **التكافؤ**: corpus ذهبي جديد (`rules_golden.jsonl`) يُثبت التطابق في ثلاث بيئات — `cargo test` أصليًا، Node (`bench.mjs`)، وداخل Chromium حقيقي (`browser_proof.mjs` عبر Playwright): **10/10 نصوص متطابقة مع مرجع بايثون في المتصفح**.

**قياسات المتصفح (HeadlessChrome 141):** فحص جملة p50 = **0.100ms**، فقرة 440 حرفًا p50 = 0.400ms، مستند 9.9K حرف (540 ملاحظة) p50 = 9.3ms. الصفحة (`index.html`) تعمل دون اتصال بعد التحميل الأول — النص لا يغادر الجهاز.

**مأزق بنية تحتية حُل أثناء البناء** (موثق للمستقبل): مكوّن rust-std للهدف wasm32 محجوب شبكيًا (static.rust-lang.org)؛ الحل: rustc 1.75 + rust-src من apt، واستعادة `Cargo.lock` الجذري لـ1.75.0 من وسم git، ثم `RUSTC_BOOTSTRAP=1 cargo -Z build-std=std,panic_abort`، وربط بـ`wasm-ld` النظامي عبر شيم يحذف أعضاء `lib.rmeta` التي يرفضها LLVM-18.

## م4.2 — المزامنة الحية (مكتملة)

`dhad/sync.py`: نقطة WebSocket ‏`/ws/sync/{doc_id}` مركّبة في `create_app` (و`dhad serve`؛ التعطيل بـ`--no-sync`). **الخادم موجّه أعمى بالتصميم**: الحمولات سلاسل معتمة تُفحص حجمًا فقط، لا تُحلَّل ولا تُسجَّل — فالتشفير طرف-إلى-طرف يعمل دون أي تغيير، وقد أُثبت باختبار يُمرّر ciphertext ويتحقق من عبوره بايتًا-ببايت ومن أن النص الصريح لا يظهر في أي حمولة. تقوية: معرفات مستندات مصادَق عليها، سقوف حجم/غرفة، أكواد إغلاق 4000-4003، أحداث حضور، وتسلسل لكل مستند.

**برهان الشبكة الحية** (uvicorn حقيقي + عميلا websockets): زمن ترحيل عملية A→خادم→B ‏**p50 = 0.487ms / p95 = 0.639ms**، ~2,000 عملية/ثانية بترحيلات متسلسلة، وتقارب بايت-ببايت بين نسختي CRDT بعد 300 عملية شبكية. اختبارات آلية (9): بث، عزل غرف، كتابة متزامنة في نفس الموضع تتقارب، ترحيل مشفر، ورفض الرسائل المشوهة.
